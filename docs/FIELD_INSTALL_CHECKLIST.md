# Field Install Checklist

- Bench power-on and serial telemetry check.
- Confirm sticker matches serial/device_id/DevEUI.
- Pair node to correct vineyard block in dashboard.
- Install soil probe at target root depth.
- Mount BME280 radiation shield 6–12" from box.
- Place lux sensor at intended canopy measurement point.
- Install optional leaf wetness paddle (if precision_plus).
- Verify first telemetry appears end-to-end in dashboard/API.
- Confirm alert logic and block mapping are correct.
